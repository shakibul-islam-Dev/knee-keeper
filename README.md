
KeenKeeper

A streamlined personal relationship management dashboard designed to help users track and maintain meaningful connections through structured interaction logging and status-based reminders.





## Tech Stack

**Client:** Next.js

Tailwind CSS

React Router

React Icons

JavaScript (ES6+)




## Key Features

Dynamic Interaction Timeline: A centralized feed that tracks "Call," "Text," and "Video" interactions with specific visual indicators for each type.

Smart Status Logic: Automatically categorizes tasks and interactions into "On Track," "Almost Due," or "Overdue" to prioritize follow-ups.

Responsive Dashboard UI: A clean, developer-centric interface built with Tailwind CSS, featuring custom input fields and buttons for a seamless user experience.
